About This project


The following are the features of this simple tasklist project.

1) The user can register/change the password using otp authentication through email in the Login page.

2) The user will receive a welcome email on successful registration.

3) The user can reach out to admin through contact.

4) The user can update their tasklist by adding or deleting the created tasks.
